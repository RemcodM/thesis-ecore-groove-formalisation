## Isabelle proofs
This directory contains the Isabelle proofs for this thesis.

### Get started
The Isabelle proofs provided with this thesis are divided over 4 Isabelle sessions, named `GROOVE`, `Ecore`, `Ecore-GROOVE-Mapping` and `Ecore-GROOVE-Mapping-Library`:

- `Ecore` consists of the formalisation of Ecore, as well as formalisations for combining type models and instance models.
- `GROOVE` consists of the formalisation of GROOVE, as well as formalisations for combining type graphs and instance graphs.
- `Ecore-GROOVE-Mapping` consists of formalisations related to mapping functions from Ecore to GROOVE and vice versa
- `Ecore-GROOVE-Mapping-Library` consists of proofs for small transformations between Ecore and GROOVE.

### Usage

In order to load and use the provided proofs, 'Graph Theory' from the Isabelle Archive of Formal Proofs is required. To install the Isabelle Archive of Formal Proofs, [follow these instructions](https://www.isa-afp.org/using.html).

Because the provided proofs consist of multiple sessions, the proofs delivered with the thesis must be installed in a similar fashion to the Isabelle Archive of Formal Proofs. Download the ZIP file with proofs and unpack these to a known directory, for example '~/Ecore-GROOVE-Proofs'. Then, add this directory to the Isabelle ROOTS file, like:
```
echo "/home/myself/Ecore-GROOVE-Proofs" >> ~/.isabelle/Isabelle2019/ROOTS
```
Restart any Isabelle session and the proofs are ready to be used.
